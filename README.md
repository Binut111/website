# Personal Portfolio Website

A responsive and interactive personal portfolio website for Bernard Juk, Robotics and Automation Engineer.

## Files

- `index.html` - Main website structure
- `styles.css` - Responsive layout, dark mode, animation, and styling
- `script.js` - Mobile menu, theme toggle, project filtering, modal details, and form validation

## Main Features

- Responsive design for mobile, tablet, and desktop
- Sticky navigation
- Mobile hamburger menu
- Dark and light theme toggle
- Smooth scrolling
- Project category filters
- Project detail modal
- Contact form validation
- Scroll reveal animation
- Self-contained static website

## How to Run

1. Download and unzip the folder.
2. Open `index.html` in any modern browser.
3. Edit the content in `index.html` to add your real email, LinkedIn, resume, and project details.

## How to Publish

Recommended options:

1. GitHub Pages
   - Create a GitHub repository.
   - Upload these files.
   - Go to Settings > Pages.
   - Select branch `main` and root folder.

2. Netlify
   - Drag and drop the full folder into Netlify.
   - Netlify will publish it as a live website.

3. Vercel
   - Import the repository.
   - Deploy as a static site.
